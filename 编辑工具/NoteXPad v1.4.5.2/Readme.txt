软件简介：
---------

    NoteXPad 是一个记事本软件，是用 win32 汇编语言来开发的。

    NoteXPad 的主程序仅有几十K大小，但它提供的功能毫不含糊，有鲜明的特色：
    它可以快速打开任何大小的文件、可以设定自动复制、自动粘贴、支持语法及
    超链接的高亮显示。目前，它仍在发展完善之中，将具备更多的功能。

    以上这些特色，使 NoteXPad 完全超越了 Windows 自带的记事本，让用户体验
    到全新功能的记事本。

    NoteXPad是一个免费软件，你可以无偿地使用它的所有功能。

    如果用户能够通过论坛或 email 及时反馈软件中存在的错误，提出好的建议，
    将被视为对作者的热心支持。如果用户能在物质上捐赠这个自由软件的开发，
    同样也是对作者的支持。

    请经常到软件主页 http://dREAMtHEATER.yeah.net 检查更新。


最小要求：
---------
    o IBM 兼容的 PC 机。
    o MS Windows 98, 2000 或后续的 Windows 版本。
    o Riched20.dll 5.30.23.1200或更高版本（你可以从微软的站点
      http://www.microsoft.com/msdownload/platformsdk/instmsi.htm 下载
      Windows Installer 来升级你的 riched20.dll。
    o 300KB的剩余磁盘空间。


主要功能：
---------

    o 可以打开任意大小的文件。
    o 自动复制，在 NoteXPad 中被选取的内容，将自动复制到剪贴板中。
    o 自动粘贴，NoteXPad 能监视其他程序所复制的文本，并将其内容及时自动
      地贴到NoteXPad 里  
    o 自动网址识别，NoteXPad 可以用蓝色来提示文本中的超链接内容。
      在 NoteXPad 中，文本中超链接格式的内容如果被点击，将会调用浏览器
      来打开相应的网址。
    o 语法高亮显示，已支持 C++，Assembly，Java 等多种编程语言（用户可以
      通过编辑 WordList.syn 来扩充支持更多的编程语言）。
    o 持续监视打开的文件，当被另一个应用程序修改时进行通知。
    o 可预览文件。
    o 识别的编码格式已包括 ANSI（默认）、Unicode、Unicode (big endian)、
      UTF-8。
    o 识别 WIN（默认）、UNIX、MAC 的文本格式，并能互相转换。
    o 能与任何文件类型进行关联，并且可以正确地恢复到以前的关联。
    o 用户可以定制其中的工具菜单，设定快捷键，轻松调用外部工具。
    o 保存最近打开文件列表，可最多保存16个文件名。 
    o 程序异常捕获，并可以发送错误报告。


操作介绍：
---------

    o 用户可以通过“查看”设置最常用设置。
    o 在“选项”中，用户可以定义 NoteXPad 的绝大部分设置。
    o 在“定制工具”中，用户可以定制快捷键，在 NoteXPad 中调用第三方工具。

    具体操作见帮助文件 NoteXPad.chm。


常见问题：
---------

1.  我在使用 NoteXPad 过程中一些功能好像不起作用，是什么原因？

　   近来有人反映他们在使用 NoteXPad 的过程中遇到一些问题，我想很可能的原
     因是他们的机器中没有安装正确版本的 RichEdit 控件。NoteXPad 使用的
     RichEdit 版本是3.0，该 DLL 包括在 Windows ME/2000/XP 和
     Windows Installer 中，你可以看看你的 SYSTEM 目录中（NT下是SYSTEM32目
     录）的 Riched20.dll 文件，如果该文件的版本号为5.30.23.1200或更高，则
     你的系统中就安装了 RichEdit 3.0。如果没有，请到微软的网站下载安装文
     件 http://www.microsoft.com/msdownload/platformsdk/instmsi.htm，它会
     自动升你机器中的 RichEdit，请根据你使用的平台来下载不同的升级文件！

2.  该如何对“定制工具”进行设置？

　   o 标题
     显示在“工具”菜单中的工具或命令的标题。在工具名称中的一个字母前放置
     一个“&”，会使这个字母成为这个工具的加速键，例如，“&Explorer”将在
    “工具”菜单中显示“Explorer”，另外你可以使用“-”来指出你想添加一个
     菜单分隔符。

     o 快捷键
　　 为这个工具指定一个快捷键，你可以直接输入一个键组合来作为快捷键，键的
     名称将显示在编辑框中。

     o 命令
　　 设置指向.exe, .com, .pif, .bat, .cmd 或其他你想运行的文件。

     o 参数
　　 设置当工具或命令运行时所传递的值。例如，参数可以是一个文件名或路径，
     你可以使用“->”按钮弹出菜单，从中选择宏参数（完整的宏参数及它们的定
     义，参见下面列表），你可以添加多个宏参数。此外，你还可以输入定制的参
     数― 命令行开关，这依赖于你使用的命令或工具。

　　 o 宏参数列表
    （均指当前打开文件）
  　 宏　　 含义
　　 %P　　仅文件路径
　　 %N　　仅文件名称
　　 %E　　仅文件扩展名
　　 %F　　完整文件名

3.  如何修正了98下中文查找的bug的?

     在查找之前，先调出中文输入法，再进行查找，可暂时解决该问题。
