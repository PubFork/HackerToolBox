Options Menu

 

Stay On Top

Make the windows on topmost

Minimize to System Tray

When the program is minimized is moved to system tray

Only One Instance

Allow to use only one instance

Enable Native API

Use NTAPI for delete key with null char

Auto Backup

Make a backup of the keys before delete

Disable Auto-Size List

Disable the Listview column's auto resizing

Disable Date/Time Column

Hide the Date/Time Column

Hide Type Column

Hide the Type column

Show CheckBox

Show the checkbox (like version 3.x)

Disable Clear List Before Scan

Don't clear the listview before start new scan

Save Window Position

Save window position before exit

Hide Keys List Button

Disable the Keys List Button in toolbar

Hide Protect Key

Hide the protect keys during scan

Disable Disk Sector Scan

Disable the disk sector scan

Customize Toolbar

Show the toolbar customization form

Enable Log

Write the internal log on "Trial-Reset.log"

Enable AERO Graphic

Enable some graphic hack for better apparence on AERO

WoW64 (x64 only)

Select the WoW64 mode:

-Auto:      automatically select scanner to use for 32bit or 64bit (best choice)

-32bit:      force all scanner to use 32bit registry and filesystem

-64bit:      force all scanner to use 64bit registry and filesystem

-Both:      force all scan to use 32bit and 64bit registry and filesystem (complete but slow)

Opacity

Set the window opacity

Toolbar Columns

Select the number of columns for the toolbar's icons (replace the old "Double column" option)

Drive Select

Select the drive to be use in the scanner that analize full disk (like ActiveMark or Sofwrap)

Autocleaning at Startup

Enable:    Enable\Disable Autocleaning

Delay:     Set the startup delay

Priority

Select the process priority

Regedit Jump Mode

Select the mode to use for jump on the regedit

Standard:    use the standard sendmessage mode (for compatibility problem)

Advanced:   use a fast jump mode (default mode)

Sort List By

Column to use for sort the listview

Nuke-Stop

Enable the clock freezing for some old time-limited protectors
