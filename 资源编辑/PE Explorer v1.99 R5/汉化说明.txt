【软件简介】

    功能极为强大的可视化汉化集成工具，可直接浏览、修改软件资源，包括菜单、对话框、字符串表等； 另外，还具备有 W32DASM 软件的反编译能力和PEditor 软件的 PE 文件头编辑功能，可以更容易的分析源代码，修复损坏了的资源，可以处理 PE 格式的文件如：EXE、DLL、DRV、BPL、DPL、SYS、CPL、OCX、SCR 等 32 位可执行程序。 该软件支持插件，你可以通过增加插件加强该软件的功能， 原公司在该工具中捆绑了 UPX 的脱壳插件、扫描器和反汇编器，非常好用。

===============================================================
      	PE Explorer: 版本历史
===============================================================

	图例:
	+ 新特性
	* 改进/修正或者 BUG 修正


 1.99 R5 (10/28/2008)

	常规: 
	* BUG 修正: 修正在 Windows Vista 下的 UI 错误。
	* BUG 修正: 修正退出 PE Explorer 时偶然发生的系统锁死。
	* 其他小的 BUG 修正和改进。

	文件头信息查看器:
	+ 在描述字段增加字段编辑器来查看或设置标记位。

	重定位表查看器:
	* BUG 修正: 不能显示位于文件资源区段后的重定位表。

	资源编辑器:
	* 在风格加载向导中增加处理器构架选项。 
	* BUG 修正: 当最后编辑的项目是菜单的分隔器时发生的错误。

	UPX 脱壳插件: 
	* 小更新。


 1.98 R2 (05/02/2006)

	* BUG 修正: 在尝试查看区段的 RAW 数据的情况下产生访问违
          例。
	* BUG 修正: 对话框编辑器不允许编辑 Windows Mobile 5 文件
          的对话框。
	* BUG 修正: 从文件中替换图标资源时发生的错误。
	* BUG 修正: 尝试播放 .AVI 资源时产生的错误。
	* BUG 修正: 显示资源以十六进制转储时发生的错误。
	* BUG 修正: 修正在使用数字小键盘输入数字时的问题。
	* BUG 修正: 修正当编辑 Delphi 图像列表时的一个错误 (透明
          图像失效，图片显示为单色位图)。
	* 其他小的 BUG 修正和改进。

 1.98 (12/01/2006)

	资源编辑器:
	+ 现在支持查看和替换 Vista 的 PNG 大图标 (256x256) 。
	* BUG 修正: 修正当显示的资源以十六进制转储时发生的错误。

	UPX 脱壳插件:
	* 现已支持对老版本 UPX (0.80 以前版本) 加壳程序的脱壳。
	* 尝试恢复脱壳后原始 PE 文件的文件头入口无效的文件。

	数据目录:
	* 添加设置到零的按钮，使删除指示器更加容易。

	反汇编器:
	* BUG 修正: 修正在 VCL 对象中搜索字段偏移量时发生的错误。
	* 其他小的改进。
