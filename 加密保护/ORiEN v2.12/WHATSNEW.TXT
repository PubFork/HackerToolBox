История создания ORiEN
~~~~~~~~~~~~~~~~~~~~~~

V2.12:
~~~~~~
+ полная версия программы выводит имя лицензионного владельца 
+ поддержка английского, немецкого и испанского языков

V2.11:
~~~~~~
В июле, на смену своему P-166MMX с 32-мя Мбайтами на борту, купил новый
компьютер. В результате, с помощью новой вычислительной мощности и была 
создана эта версия защиты ORiEN. Надеюсь, что новая версия Вам понравиться!

Новое:
+ встроенная функция генерации ключа для привязки
+ встроенная функция отправки сообщений об ошибке
+ опционален язык диалога запроса пароля
+ заданы несколько ключей HASP
+ сменили версию компилятора Delphi с 3-й на 7-ю
+ теперь ORiEN поддерживает темы рабочего стола Windows XP
Модификация:
* полностью корректная работа диалога запроса пароля загрузчика
* корректная работа защищенных приложений Windows XP
* при наступлении демонстрационного ограничения, с отключенной опцией
  "Вечное демо", защита, как и положено, завершает работу приложения
* не удавалось задать ключ HASP по причине неправильной обработки пароля ключа
* перестановки и усовершенствования на форме
* исправлены ошибки с подсветкой некоторых элементов
* исправлен порядок переключения по форме с помощью табуляции
* расширена документация
* файл конфигурации теперь с расширением ".ocf"
Прочее:
> проведено полное тестирование под Windows XP, Windows 2000 и Windows 98
> проверена совместимость с программами Visual Basic, Visual FoxPro и 
  приложениями PowerBuilder

V2.10:
~~~~~~
Новое:
+ привязка к HASP-ключу (используется стандартная библиотека "haspms32.dll")
+ опционально сжатие защищаемой программы
+ опциональна многоуровневая защита с помощью ORiEN
+ возможность не прерывать работы при наступлении демонстрационного события
+ опционален показ или сокрытие пароля при вводе в диалог загрузчика
+ опционально количество полей ввода пароля в загрузчике
+ автоматическая генерация паролей для привязки
+ при первом старте самостоятельно регистрируемся в меню кнопки "Пуск"
+ показываются параметры командной строки, передаваемые загрузчику 
  при пробном запуске
+ добавлена детекция упаковщиков ASPACK и PE-CRYPT
+ продвинутая система сохранения профилей
Модификация:
* корректная работа под Windows 2K/XP:
  - исправлена ошибка загрузки перекрестной ссылки на другую библиотеку
  - корректно выводится заголовок окна затребования пароля под Windows 2K/XP
  - диалоговое окно ввода пароля теперь можно двигать
* внимательно изучены дампы защищенных файлов - теперь все бреши закрыты
* корректная установка защиты на большие файлы с плохой упаковкой
* корректная обработка ситуации, когда при загрузке возникала ошибка импорта
  какой-либо функции из библиотеки
* расширена защита от отладчиков и дизассемблеров и сграбления памяти
* подсвечивались элементы при загрузке языкового файла
* резервная копия файла создавалась не проверив формат файла
* переработана поддержка командной строки
* расширил детекцию UPX

V2.01:
~~~~~~
* убрано установленное имя файла для защиты, которое забыл убрать после тестов
* подкорректирован номер версии ;)
