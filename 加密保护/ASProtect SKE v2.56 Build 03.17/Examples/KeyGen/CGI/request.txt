
This example shows how to execute a CGI module that generates
a registration key for the example project (reg_trial.aspr2)

Constants (p1, p3, p6, p7) from your project (.aspr2 file,
Activation Keys section)

C1    - p1
C3    - p3
C6    - p6
C7    - p7

RegName         - Registration Name
HardwareID      - Hardware Identifier
ExpirationDate  - Key expires in N day later
EncryptionConst - Encryption constant
ModeID          - Mode number (1..15)
SignatureBits   - Signature size in bits
