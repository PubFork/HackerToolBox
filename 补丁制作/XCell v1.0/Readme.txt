软件简介：
---------

    XCell 是一个专业的补丁制作软件，全部功能高度集成在一个程序之内，是一个
    纯粹的绿色软件。它可以对文件名和文件大小相同的文件进行逐字节比较，然后
    高速快捷地生成文件对文件，目录对目录，文件对目录之间同文件的补丁制作，
    具有补丁批处理制作的能力；同时生成的补丁程序非常之小，方便网上发布，并
    对作者的信息进行了有效保护。该软件特别适合汉化人制作汉化补丁批处理程序。

    XCell 是一个免费软件，你可以无偿地使用它的所有功能。

    如果用户能够通过论坛或email及时反馈软件中存在的错误，提出好的建议，将
    被视为对作者的热心支持。如果用户能在物质上捐赠这个自由软件的开发，同样
    也是对作者的支持。

    请经常到软件主页 http://NoteXPad.yeah.net 检查更新。


最小要求：
---------
    o IBM 兼容的 PC 机。
    o MS Windows 98, 2000或后续的 Windows 版本。
    o 350KB的剩余磁盘空间。


功能特点：
---------

    主程序（XCell）：

    o 单个文件之间的补丁制作；
    o 对整个目录及其下面的子目录内的文件进行补丁批处理制作；
    o 向导型窗体界面，操作简单、明了，只需三步即可完成补丁文件的生成；
    o 可定制补丁制作信息；
    o 生成的补丁文件是单一的可运行文件，无需任何其他辅助文件的支持；
    o 可定制生成的运行文件的程序图标；
    o 生成的补丁文件体积很小，非常适合网上发布。

    前台程序（生成的补丁程序）:

    o 运行初始化时，自动判断当前目录是否为补丁应用目录；
    o 对文件进行补丁动作之前检查文件的完整性（CRC32校验）；
    o 可备份原始文件；
    o 不修改文件的创建时间以及文件属性。
